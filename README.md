# 概要
Bve5.8/6.0用　近カコ・近ヒネ・近モリ・近ナラ103系車両データです。車両指定は連絡なしで自由に行えます。  

# ファイル構成  
```
Scenarios - △12△  
　　　　 - Rock_On  
　　　　 - GeneralAtsPlugin  
```
となるよう配置してください。  

# 導入が必要なもの
JR線車両パック(Rock_On様)  

# SpecialThanks
